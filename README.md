#  Sistema de Gestor de Expedientes

Este proyecto consiste en una aplicación web para el control de expedientes clínicos, desarrollado como parte de mi servicio social en el IMSS. Incluye:

- 📱 **Frontend**: Angular + TailwindCSS
- 🧠 **Backend**: NestJS + PostgreSQL

| Backend     | [Repositorio backend](https://github.com/Brashan7u7/SGE-Backend-.git)   |

---

## 🚀 Funcionalidades principales

- Registro y consulta de expedientes.
- Control de préstamos de documentos.
- Panel de administración para usuarios Arimac y Administradores.
- Observaciones internas por expediente.
- Autenticación y control de sesiones.



---

## ⚙️ Tecnologías utilizadas

- Angular · NestJS · PostgreSQL · Docker · TailwindCSS
